Account Security and Authentication Settings

This document outlines the security and authentication settings available for your account.

Two-Factor Authentication

Add an extra layer of security to your account.

How to use: Enable this feature to require a second form of verification (e.g., a code sent to your phone) when logging in.

Password Reset

Change your account password.

How to use: Click on "Reset Password" to initiate the password change process. Follow the prompts to set a new, strong password for your account.

API Key

Your API key for accessing our services.

How to use: Use this key to authenticate API requests to our services. Keep this key secret and secure.

Additional action: Click "Regenerate" to create a new API key if you suspect your current key has been compromised.

Session Timeout

Set the duration of inactivity before automatic logout.

Current setting: 30 minutes

How to use: Adjust this setting to balance security with convenience. A shorter timeout increases security but may require more frequent logins.

Login History

View your recent login activity.

How to use: Click "View Login History" to see a list of recent logins to your account, including date, time, and location information.

Audit Logs

View a log of all actions taken in your account.

How to use: Click "View Audit Logs" to access a detailed record of activities within your account. This can help you monitor for unauthorized access or suspicious activities.
